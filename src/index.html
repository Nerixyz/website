Current Song2

An application allowing (mostly) streamers to display the currently playing song. It uses Windows' native media transport control to show instant updates of songs and play progress. The overlay is fully customizable through CSS and JS and ships in a single executable.

Instagram Private API

This is a TypeScript library which is using Instagram's Android-API to allow the upload of media or viewing of profiles just to name a few. I'm one of the contributors.
